# CYS Academy Student Result Portal 📊

A professional, secure web application for students to view their examination results and access question papers. Built with HTML, CSS, and JavaScript - no backend required, perfect for GitHub Pages hosting.

## ✨ Features

- **🔐 Secure Access**: Each student has a unique 6-character index number that's never displayed on the interface
- **📱 Mobile Responsive**: Beautifully adapts to all screen sizes
- **🎯 Exam Specific**: Currently configured for "Sequences and Functions" examination
- **📊 Discipline Tracking**: Visual progress bar showing discipline scores (customizable per student)
- **📄 Google Drive Integration**: Direct links to view question papers
- **🎨 Modern UI**: Glass-morphism design with professional, clean aesthetics
- **🔄 Simple Navigation**: Easy flow from index entry → results → question paper

## 💻 Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Glass-morphism)
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts (Inter)
